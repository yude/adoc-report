## Install
* Install applications listed below.
  * TeX Live
  * pandoc
  * pygments
    ```
    pip install pygments
    ```
  * pandoc-minted
    ```
    pip install pandoc-minted
    ```
* Add `bdoc`, TeX Live, pandoc, `minted_filter.py` to your `PATH`.

## How to use
```shell
$ bdoc hoge.md
```
