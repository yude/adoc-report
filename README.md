## Install
* Install applications listed below.
  * TeX Live
  * pandoc
    ```bash
    # Arch Linux
    $ sudo pacman -S pandoc
    # macOS
    $ brew install pandoc
    ```
  * AsciiDoc
  * Asciidoctor
* Add `bdoc`, TeX Live, pandoc to your `PATH`.

## How to use
```bash
$ bdoc hoge.adoc
```
